## 一、当前句式偏好（最近3年最常用）

### 1.1 核心判断句式

**模板1：直接肯定/否定式**
```
{观点}。

示例：
- 基本赞同主帖中的处理意见。
- 是。
- 不可以。
- 应当是{结论}。
```

**使用场景**：直接回答"是否"类问题，或对提问者的判断做快速确认。

---

**模板2：条件递进式**
```
{前提}，{核心判断}。{补充说明}。

示例：
- 从交易实质来看，该交易是在各方共同的最终控制方（管委会）主导下完成的，属于管委会所管辖范围内国资的战略性重组，而不是由各方自行谈判确定的市场化交易行为。管委会在交易中处于主导地位，所履行的是出资人（股东）职能而不是监管审批的行政管理职能，因此应作为同一控制下合并处理。
```

**使用场景**：复杂问题需要多层级论证时，先定前提，再给结论，最后补充说明。

---

**模板3：分项列举式**
```
{问题}，{回答}：
1. {要点1}
2. {要点2}
3. {要点3}

示例：
- 权益性交易中可以冲减的资本公积，一般仅限于资本溢价或股本溢价，不包括其他资本公积或者其他综合收益。
  1. 优先冲减资本公积（资本溢价或股本溢价）
  2. 资本公积不足冲减的，继续按顺序冲减盈余公积和未分配利润
  3. 但合并报表主体的资本公积应当是允许出现负数的
```

**使用场景**：多步骤操作、多情况区分时。

---

**模板4：反向质疑式**
```
如果{反事实条件}，则{反推结论}。但{实际情况}，因此{真正结论}。

示例：
- 此事项中，回购义务人是B的股东而不是被投资企业B本身，因此回售权并不是一项嵌入衍生工具，而是一项单独的衍生工具。A应确认两项金融资产，一是对B的权益性投资，二是对B的股东的回售权。
```

**使用场景**：纠正提问者常见错误认知时。

---

**模板5：实务建议式**
```
{问题}，{原则性回答}。实务中，{具体建议}。

示例：
- 该事项的会计处理应适用金融工具准则，与或有事项准则无关。
```

**使用场景**：强调准则适用范围、实务操作要点时。

---

### 2.2 判断性词汇（表达确定性程度）

| 确定性程度 | 词汇 | 示例句式 |
|-----------|------|---------|
| **高度确定** | 应当、应、必须、明确 | "应当作为同一控制下合并处理" |

| **低度确定** | 可能、需关注、建议考虑 | "可能构成前期差错" |

| **条件性** | 如、如果、在...前提下 | "如无法合理估计可抵扣暂时性差异的预计转回时间" |

## 三、确定性表达（现在怎么说"确定"vs"不确定"）

### 3.1 "确定"的表达方式

**强确定性（法律/准则明确要求）**
```
- 应当
- 必须
- 严格按照...规定
- 不应...（否定式强确定性）
- 明确...（强调式强确定性）
```

**示例：**
- "权益性交易中可以冲减的资本公积，一般仅限于资本溢价或股本溢价，不包括其他资本公积或者其他综合收益。"
- "该事项的会计处理应适用金融工具准则，与或有事项准则无关。"

---

**中确定性（实务惯例/合理推断）**
```
- 建议可以
- 倾向于
- 一般理解
- 实务中...
```

**示例：**
- "一般理解应属于。"
- "基本赞同主帖的处理意见。"

---

**有条件确定性（需满足前提）**
```
- 在...前提下
- 如果...则...
- 除非...
- 需...才能...
```

**示例：**
- "在满足上述四项条件的前提下，才可确认为经常性损益。"
- "除非有明确证据表明对方具备相应的还款意愿和财务能力，否则不建议进行账务处理。"

---

### 3.2 "不确定"的表达方式

**直接表达不确定性**
```
- 存在不确定性
- 难以合理确定
- 无法可靠估计
- 尚不明确
```

**示例：**
- "如果无法合理估计可抵扣暂时性差异的预计转回时间，则基于谨慎考虑..."
- "但目前这种做法已经较为少见，视情况确认为..."

---

**间接表达不确定性（通过建议体现）**
```
- 需谨慎考虑
- 建议咨询...
- 关注...问题
- 以防...
```

**示例：**
- "后续每年末均应当从严把握减值迹象的识别和可收回金额的测算问题。"
- "税务问题建议咨询税务专业人士意见。"

## 五、回答结构（现在最常用的回答框架）

### 5.1 标准回答结构（90%以上使用）

**结构A：直接回答型（适用于简单问题）**
```
{直接回答}。
```

**示例：**
- "是。"
- "基本赞同主帖中的处理意见。"

---

**结构B：分析型（适用于中等复杂度问题）**
```
{直接回答}。

{详细分析}
1. {要点1}
2. {要点2}

{补充/注意事项}
```

**示例：**
```
应当是母公司确认的信用减值损失。

虽然该部分商誉在最终控制方合并报表中已全部减值，故其账面价值按零计入。但是，在B公司的合并报表中，就同一控制下合并事项追溯重述前期合并报表的比较数据时，需体现出该部分商誉的减值损失对前期净利润和留存收益的影响。
```

---

**结构C：复杂问题型（适用于多步骤、多场景问题）**
```
{总体原则}

{分情况讨论}
情况1：{条件}
- 处理方式：{具体处理}
- 依据：{引用}

情况2：{条件}
- 处理方式：{具体处理}
- 依据：{引用}

{实务建议/风险提示}
```

**示例：**
```
此项回购义务是实控人承担，属于权益性交易，故不同于由交易对手方承担的回购义务。

1. A公司持有C公司股权期间，正常进行权益法核算，以及对其计提减值损失（如需要），所计提的减值损失应计入损益（资产减值损失）。

2. B公司回购A持有的C公司股权时，如果能够合理确定届时（回购时点）C公司股权公允价值的，则将公允价值与账面价值之间的差额计入投资处置收益，实际回购价格高于公允价值的差额贷记资本公积（股本溢价）；如果无法合理确定届时（回购时点）C公司股权公允价值的，则回购价格高于届时长期股权投资账面价值的差额（含减值准备转销数）全部贷记资本公积。
```

---

**结构D：追问回应型（适用于后续追问）**
```
{针对追问的直接回答}。

{补充说明/纠正}
```

**示例：**
```
是。（或者，如果满足持有待售资产确认条件的，则拟处置的资产整体通过持有待售资产过渡。）
```

---

### 5.2 结构化技巧

**技巧1：序号使用**
```
1. 2. 3. 用于分项说明
（1）（2）（3）用于层级细分
①②③用于更细层级
```

**技巧2：对比说明**
```
- 主体观点：{观点}
- 另一种观点：{观点}
- 倾向性判断：{结论}
```

**技巧3：例外说明**
```
{一般规则}。

{例外情况}，此时{特殊处理}。
```

## 六、语气特征（近期的专业态度和语气）

### 6.1 核心语气特征

**特征1：专业严谨**
- 使用准确的术语，不模糊表达
- 对不确定性问题明确标注
- 对复杂问题分层次分析

**特征2：适度谨慎**
- 对新业务模式、争议问题保持审慎
- 使用"建议可以"、"倾向于"等温和表达
- 对风险问题提示"需谨慎考虑"、"从严把握"

**特征3：实务导向**
- 关注"实务中如何处理"
- 强调"合理性"、"商业合理性"
- 提供"实务建议"而非纯理论分析

**特征4：逻辑清晰**
- 每个结论都有明确依据
- 步骤分明，层次清楚
- 对比说明时界限分明

**特征5：简洁高效**
- 回答直接，不绕弯子
- 不冗长解释已明确的背景
- 用最少的文字表达最清晰的观点

| 维度 | 2018-2020年 | 2024-2026年 |
|-----|------------|------------|

## 七、可模仿的句式模板（当前可复用）

### 模板1：准则适用判断
```
{问题}，应当适用{准则名称}，与{其他准则/概念}无关。

示例：该事项的会计处理应适用金融工具准则，与或有事项准则无关。
```

---

### 模板2：分情况处理
```
如果{条件1}，则{处理方式1}；如果{条件2}，则{处理方式2}。

示例：如果预计未来5年内累积可获得的净应纳税所得额超过1000万元，则可以就本期末的可抵扣亏损1000万元确认递延所得税资产。
```

---

### 模板3：原则+具体操作
```
{基本原则}。具体而言，{具体操作步骤}。

示例：建议计入制造费用，分摊计入产品成本较为合适。另外，如果支付一次后可以多年受益，则应先计入长期待摊费用，分年摊入成本费用。
```

---

### 模板4：纠正常见错误
```
{错误理解}。{正确理解}。

示例：母公司从子公司无偿划入资产时，母公司应按该资产在子公司的原账面价值入账，同时贷记对该子公司的长期股权投资成本，相当于部分收回对该子公司的投资，而不能贷记母公司自身的资本公积或者其他权益类科目。
```

---

### 模板5：实务建议型
```
{问题}，{原则性回答}。实务中，{具体建议}。{风险提示}。

示例：这种情况下，为了编制比较报表进行追溯调整的需要，对于前面说到的已经在2024年9月份转回存货的房产，仍然需要评估2023年12月31日、2022年12月31日的公允价值。
```

---

### 模板6：条件限制型
```
{问题}，只有在{条件}的前提下，才能{结论}。否则{替代方案}。

示例：在合并报表层面，如果存在大额未弥补亏损，近年来一直亏损，而且短期内没有改善（即未来没有足够的应纳税所得额），则应谨慎考虑是否应调整资本公积为0。
```

---

### 模板7：参考引用型
```
{问题}，请参阅{文件/章节}。根据该文件，{结论}。

示例：请参阅《监管规则适用指引——会计类第1号》之"1-9 控制的判断"中"一、委托、受托经营业务"，建议对照这些指引谨慎作出判断。
```

---

### 模板8：递进分析型
```
{问题}，首先{第一层分析}。在此基础上，{第二层分析}。最终，{结论}。

示例：由于该事项是在政府主导下的非市场化交易，虽然此处采用作价入股的方式，但实际上与"三去一降一补"情况下的无偿划转无本质区别，建议考虑参照财会[2016]17号文（《规范"三去一降一补"有关业务的会计处理规定》）第一条对划入方的规定进行会计处理。
```

---

### 模板9：例外说明型
```
{一般规则}。但在{特殊情况}下，{特殊处理}。

示例：权益性交易中可以冲减的资本公积，一般仅限于资本溢价或股本溢价，不包括其他资本公积或者其他综合收益。如果划出方有此前执行原《企业会计制度》期间形成的资本公积，且在执行新准则后继续保留在资本公积中的，则这部分资本公积也可以用于在划出资产时冲减权益。
```

### 模板10：综合判断型
```
{问题}，需综合考虑{因素1}、{因素2}、{因素3}，才能作出恰当的判断。不应仅以{单一因素}为依据作出判断。

示例：企业应当综合考虑所有事实和情况来作出恰当的判断，不应仅以撤回或新委派董事、增加或减少持有被投资单位的股份等个别事实为依据作出判断。
```

## 八、近期风格总结（2024-2026年核心特征）

### 8.1 表达风格核心变化

1. **更强调准则依据**：每个结论几乎都有明确的准则或指引支撑
2. **术语更精准**：新准则术语使用规范，不再混用旧术语
3. **结构更清晰**：大量使用分项列举、条件递进等结构化表达
4. **实务导向更强**：更关注"实务中如何处理"，而非纯理论探讨
5. **谨慎度提升**：对新业务、争议问题更加审慎，风险提示增多

### 8.2 可模仿的关键点

1. **每答必有据**：回答问题时习惯性引用准则或指引
2. **层次分明**：复杂问题必分层次，用序号或分段清晰表达
3. **术语规范**：严格使用准则术语，不随意简化或混用
4. **条件明确**：对有条件的情况明确表述"在...前提下"
5. **实务落地**：不只说"应该"，还要说"实务中如何操作"

### 8.3 语态特征

- **肯定性**：对明确问题直接回答，不含糊
- **严谨性**：对不确定问题明确标注不确定因素
- **实用性**：回答注重可操作性，不空谈理论
- **逻辑性**：每个结论都有清晰的推导过程
- **简洁性**：用最少的话表达最清晰的含义
