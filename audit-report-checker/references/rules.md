# 审计报告勾稽规则库（L2 内置）

本文件是审计报告核查 skill 的核心规则库，用自然语言描述三类勾稽关系：报表间恒等式、报表与附注科目对照、附注内部勾稽。L1 代码层（`scripts/calculator_rules.py`）负责执行稳定恒等式的算术，本文件为 L2 提供对照表和规则说明，由 Claude 理解后定位取数，calculator 执行验算。

**Claude 何时读这个文件**：
- 检查表注勾稽时（"应收账款的附注汇总数是否等于报表数"）——查 B 部分对照表
- 检查附注变动表时（"固定资产期初+增加-减少是否等于期末"）——查 C 部分规则
- 用户问"你们覆盖了哪些科目"时——A/B/C 三部分均可作为说明清单
- 用户想新增规则时——参照本文件格式写入 `user_rules.md`

---

## A. 报表间勾稽规则

本部分规则已在 `scripts/calculator_rules.py` 代码化为参数化函数（L1），这里列出清单便于用户理解覆盖范围。Claude 调用 calculator 执行验算，不手算。

### A1. 资产负债表

| 规则 | 表达式 | 说明 |
|---|---|---|
| 流动+非流动=资产总计 | 流动资产合计 + 非流动资产合计 = 资产总计 | 期末/期初各验一次 |
| 流动+非流动=负债合计 | 流动负债合计 + 非流动负债合计 = 负债合计 | 期末/期初各验一次 |
| 资产=负债+权益 | 资产总计 = 负债合计 + 所有者权益合计 | 会计第一恒等式，必验 |
| 权益结构 | 实收资本+资本公积+盈余公积+未分配利润+其他综合收益等 = 所有者权益合计 | 含少数股东权益时归属母公司+少数股东=合计 |

### A2. 利润表

| 规则 | 表达式 | 说明 |
|---|---|---|
| 营业利润推导链 | 营业收入 − 营业成本 − 税金及附加 − 销售费用 − 管理费用 − 研发费用 − 财务费用 + 其他收益 + 投资收益 + 公允价值变动收益 + 信用减值损失 + 资产减值损失 + 资产处置收益 = 营业利润 | 各项正负号按报表口径，损失以负号列示 |
| 利润总额 | 营业利润 + 营业外收入 − 营业外支出 = 利润总额 | |
| 净利润 | 利润总额 − 所得税费用 = 净利润 | |
| 其他综合收益 | 净利润 + 其他综合收益的税后净额 = 综合收益总额 | |
| 每股收益 | 归属于母公司净利润 ÷ 加权平均股数 = 基本/稀释每股收益 | 数量级核对 |

### A3. 现金流量表

| 规则 | 表达式 | 说明 |
|---|---|---|
| 三类净额合计 | 经营活动现金流量净额 + 投资活动现金流量净额 + 筹资活动现金流量净额 = 现金及现金等价物净增加额 | 核心恒等式 |
| 含汇率变动 | 经营净额 + 投资净额 + 筹资净额 + **汇率变动对现金的影响** = 净增加额 | 跨国/涉外企业现金流量表含此项目，提取 statements 时务必包含"汇率变动对现金的影响"项，否则会产生误报 |
| 期初期末衔接 | 期初现金及现金等价物余额 + 净增加额 = 期末现金及现金等价物余额 | 与资产负债表"货币资金"核对（口径可能含受限资金，差异需说明）|
| 补充资料间接法 | 净利润 + 调节项 = 经营活动现金流量净额 | 调节项含资产减值准备、折旧、摊销、财务费用、投资损失、递延所得税、营运资金变动等 |

### A4. 所有者权益变动表

| 规则 | 表达式 | 说明 |
|---|---|---|
| 期末=期初+变动 | 期初余额 + 本期增减变动各项 = 期末余额 | 各列分别验证（综合收益总额、股东投入减少、利润分配等）|
| 各项目横向平衡 | 各行本年金额 = 综合收益总额 + 股东投入和减少 + 利润分配 + 股东权益内部结转 各列之和 | |
| 与资产负债表核对 | 权益变动表期末余额 = 资产负债表期末所有者权益各项 | 各科目逐项核对 |

## B. 表注勾稽对照表（核心）

报表项目应当等于附注中该科目披露的构成合计。下表覆盖 41 个主要科目，Claude 按此对照表在附注中定位汇总数，calculator 比对报表数与附注数。差异超容差（0.01）记为"异常"（黄），提示人工复核。

> **使用说明**：附注披露口径多样（按类别、按账龄、按性质等），Claude 需根据附注章节标题语义判断该科目附注的汇总数取自哪个明细表。下表"附注勾稽关系"列给出最常见的取数路径，遇到变体由 Claude 理解上下文调整。

### B1. 流动资产

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 货币资金 | = 附注"货币资金"明细合计（库存现金 + 银行存款 + 其他货币资金） | 注意"使用受限的货币资金"单列项，与现金流量表口径可能不同 |
| 交易性金融资产 | = 附注"交易性金融资产"按类别合计（衍生金融资产 + 其他交易性金融资产等） | 区分"指定为以公允价值计量且其变动计入当期损益"和"交易性" |
| 应收票据 | = 附注"应收票据"按种类合计（银行承兑汇票 + 商业承兑汇票） | 注意已背书贴现未终止确认的部分 |
| 应收账款 | = 附注"应收账款"账面余额 − 坏账准备 | 区分应收账款 vs 应收票据 vs 应收款项融资；按账龄/客户分部的账面余额合计后减坏账准备合计 |
| 应收款项融资 | = 附注"应收款项融资"账面价值（公允价值） | 既以公允价值计量又兼具收取合同现金流，列报特殊 |
| 预付款项 | = 附注"预付款项"按账龄合计 | 账龄 1 年以内/1-2 年/2-3 年/3 年以上 |
| 其他应收款 | = 附注"其他应收款"按款项性质合计 − 坏账准备 | 含押金保证金、代垫款项、员工借款等；坏账准备单独披露 |
| 存货 | = 附注"存货"各项目合计（原材料 + 在产品 + 库存商品 + 委托加工物资 + 发出商品 + 周转材料等） − 存货跌价准备 | 注意"合同履约成本"是否单列；跌价准备作为抵减项 |
| 合同资产 | = 附注"合同资产"按类别合计（已完工未结算 + 其他） | 区分"合同资产"与"应收账款"：是否有无条件收款权 |

### B2. 非流动资产

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 长期股权投资 | = 附注"长期股权投资"按被投资单位明细合计 | 成本法/权益法分别披露；与合营联营企业投资明细核对 |
| 其他权益工具投资 | = 附注"其他权益工具投资"按项目合计 | 指定为以公允价值计量且其变动计入其他综合收益的权益投资 |
| 固定资产 | = 附注"固定资产"账面价值（原价 − 累计折旧 − 减值准备） | 房屋建筑物/机器设备/运输工具/电子设备分项明细 |
| 在建工程 | = 附注"在建工程"各项目合计 − 减值准备 | 含工程物资；注意"在建工程"转入"固定资产"的勾稽 |
| 无形资产 | = 附注"无形资产"账面价值（原价 − 累计摊销 − 减值准备） | 土地使用权/软件/专利/商标分项 |
| 商誉 | = 附注"商誉"账面原值 − 减值准备 | 商誉减值测试相关披露需对应；合并报表才有商誉 |
| 长期待摊费用 | = 附注"长期待摊费用"各项目合计 | 租入固定资产改良支出等；摊销年限需合理 |
| 递延所得税资产 | = 附注"递延所得税资产"按暂时性差异明细合计 | 可抵扣暂时性差异 × 适用税率；注意"递延所得税费用"勾稽 |

### B3. 流动负债

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 短期借款 | = 附注"短期借款"按种类合计（信用借款 + 保证借款 + 抵押借款 + 质押借款） | 注意已到期未偿还部分披露 |
| 应付账款 | = 附注"应付账款"按账龄合计 | 与预付账款是否重分类调整 |
| 预收款项 | = 附注"预收款项"按性质/项目合计 | 新准则下多用"合同负债"列报，预收款项仅含增值税等 |
| 合同负债 | = 附注"合同负债"按业务/客户合计 | 已收客户对价应转让商品/服务的义务 |
| 应付职工薪酬 | = 附注"应付职工薪酬"期末余额 | 期初 + 本期增加 − 本期减少 = 期末，三项均有 |
| 应交税费 | = 附注"应交税费"各税种合计（增值税 + 企业所得税 + 个人所得税 + 房产税 + 城建税等） | 印花税等通过税金及附加核算，不在应交税费列示 |
| 其他应付款 | = 附注"其他应付款"按款项性质合计（应付利息 + 应付股利 + 其他） | 区分应付利息、应付股利和其他应付款 |
| 一年内到期的非流动负债 | = 附注"长期借款/租赁负债/应付债券"中一年内到期部分 | 长期负债重分类部分，需与长期负债附注交叉核对 |

### B4. 非流动负债

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 长期借款 | = 附注"长期借款"按种类合计（扣除一年内到期部分） | 注意一年内到期部分重分类 |
| 租赁负债 | = 附注"租赁负债"按租赁类型合计 | 新租赁准则下；区分融资租赁/经营租赁列报口径 |
| 预计负债 | = 附注"预计负债"按形成原因合计（未决诉讼 + 产品质量保证 + 债务担保等） | 期初 + 本期增加 − 本期减少 = 期末 |
| 递延所得税负债 | = 附注"递延所得税负债"按暂时性差异明细合计 | 应纳税暂时性差异 × 适用税率 |

### B5. 所有者权益

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 股本/实收资本 | = 附注"股本"按股东明细合计 | 有限公司列"实收资本"，股份公司列"股本"；与股东出资明细核对 |
| 资本公积 | = 附注"资本公积"明细合计（股本溢价/资本溢价 + 其他资本公积） | 区分资本溢价和其他资本公积 |
| 盈余公积 | = 附注"盈余公积"明细合计（法定盈余公积 + 任意盈余公积） | 法定盈余公积累计达注册资本50%可不再提取 |
| 未分配利润 | = 附注"未分配利润"本期变动（期初 + 本期净利润 − 提取盈余公积 − 利润分配 = 期末） | 与利润分配表/权益变动表交叉核对 |

### B6. 利润表项目

| 报表科目 | 附注勾稽关系 | 说明 |
|---|---|---|
| 营业收入 | = 附注"营业收入"按类别/地区分部合计 | 主营业务收入 + 其他业务收入；按行业/地区/产品分部披露 |
| 营业成本 | = 附注"营业成本"按类别分部合计 | 与营业收入分部口径一致 |
| 税金及附加 | = 附注"税金及附加"各税种合计（消费税 + 城建税 + 教育费附加 + 房产税 + 印花税 + 土地使用税等） | 注意"税金及附加"vs"应交税费"口径区别 |
| 销售费用 | = 附注"销售费用"各项目合计（职工薪酬 + 折旧 + 差旅 + 广告费 + 运输费等） | 按性质披露 |
| 管理费用 | = 附注"管理费用"各项目合计（职工薪酬 + 折旧摊销 + 办公费 + 中介费 + 研发费用等） | 注意研发费用是否单独列示 |
| 研发费用 | = 附注"研发费用"明细合计（费用化研发支出 + 研发人员薪酬 + 折旧 + 其他） | 资本化部分计入开发支出/无形资产 |
| 财务费用 | = 附注"财务费用"明细合计（利息支出 − 利息收入 + 汇兑损益 + 手续费等） | 利息支出资本化部分不在财务费用 |
| 所得税费用 | = 附注"所得税费用"合计（当期所得税 + 递延所得税） | 与递延所得税资产/负债变动勾稽；有效税率核对 |

## C. 附注内勾稽规则

附注内部表格的勾稽关系，主要由横加竖加验算覆盖（详见 `structure_annotation.md`），本部分列出常见模式供 Claude 判断时参考。

### C1. 变动表 reconcile

附注中常见的"期初余额 + 本期增加 − 本期减少 = 期末余额"格式，适用科目：

| 科目 | 变动项说明 |
|---|---|
| 固定资产 | 期初原价 + 本期购置 + 在建工程转入 + 企业合并增加 − 处置/报废/折旧 = 期末原价；累计折旧、减值准备单独变动表 |
| 在建工程 | 期初 + 本期增加 − 转入固定资产 − 其他减少 = 期末 |
| 无形资产 | 期初原价 + 本期增加 − 本期处置 = 期末；累计摊销单独变动表 |
| 长期待摊费用 | 期初 + 本期增加 − 本期摊销 = 期末 |
| 长期股权投资 | 期初 + 本期增加（追加投资/损益调整/其他综合收益）− 本期减少（处置/分红）= 期末 |
| 应付职工薪酬 | 期初余额 + 本期增加（计提）− 本期减少（支付）= 期末余额；按项目分行（工资/奖金/社保/公积金/辞退福利等）|
| 应交税费 | 期初 + 本期应交 − 本期已交 = 期末；按税种分行 |
| 预计负债 | 期初 + 本期增加 − 本期减少（支付/转回）= 期末 |
| 商誉 | 期初 + 本期增加（企业合并）− 本期减少（处置）= 期末；减值准备单独变动 |

**验算方法**：calculator 的 `reconcile` 子命令，传入期初/增加/减少/期末四列，验证 期初 + 增加 − 减少 = 期末。

### C2. 坏账准备/减值准备变动表

同 C1 格式：期初余额 + 本期计提 − 本期转回/核销 = 期末余额。适用：
- 应收账款坏账准备
- 其他应收款坏账准备
- 存货跌价准备
- 长期股权投资减值准备
- 固定资产/在建工程/无形资产/商誉减值准备
- 合同资产减值准备

### C3. 常见横加模式

附注表格中列与列之间的加减关系，由 Claude 判断后 calculator 验算：

| 模式 | 表达式 | 典型科目 |
|---|---|---|
| 账面余额 − 坏账准备 = 账面价值 | 应收账款、其他应收款、应收票据、应收款项融资、合同资产 | 坏账准备可作为单独列或单独明细表 |
| 原价 − 累计折旧 − 减值准备 = 账面价值 | 固定资产 | 折旧和减值分列 |
| 原价 − 累计摊销 − 减值准备 = 账面价值 | 无形资产 | |
| 原价 − 累计折旧 = 账面价值 | 投资性房地产（成本模式）| |
| 期初 + 本期增加 − 本期减少 = 期末 | 变动表通用 | 见 C1 |

**不应横加的情况**（详见 `structure_annotation.md`）：
- 年末列与年初列属于不同时点，不能混合横加（如"年末余额-账面余额"和"年初余额-账面余额"是两个独立时点）
- 百分比列（损失率、计提比例）不参与金额加减
- 递延所得税表中"暂时性差异"与"递延所得税资产/负债"是乘税率关系，不是加减
- 按分类维度分列的表格（如按账龄分列金额）列间无加减关系
