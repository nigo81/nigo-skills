# 工商数据 52 维度字段映射

## 字段处理说明

### 数据清洗

- **电话归一化**：拆分多值（逗号/分号分隔），去区号，统一手机格式（11位）
- **邮箱归一化**：提取域名，排除公共邮箱（qq.com、163.com 等）
- **地址归一化**：去 HTML 标签、空格、标点、邮编、省市后缀，保留核心地址
- **姓名归一化**：去 HTML 标签、空格

### 特殊处理

- **法人变更.xlsx**：含 HTML 标签，使用 `strip_html()` 清理
- **持股比例字段**：equity API 中为 `czbl`，非 `investRatio`
- **公共邮箱域名**：排除 qq.com、163.com、126.com、sina.com、sohu.com、hotmail.com、gmail.com、outlook.com、foxmail.com 等
